## What i used?

- node
- mongoose
- express

## Installation

```bash
# clone the project
$ git clone https://github.com/ridvanaltun/node-restful-api.git

# navigate to the project
$ cd node-restful-api

# install dependencies
$ npm install
```

## Usage

```bash
# starts mongodb server on your local
$ mongod

# starts node server on 3000
$ npm run start
```
